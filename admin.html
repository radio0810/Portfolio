BSAFE Bingo – Admin

Use this page to control the bingo game shown on the Visix display.

How this page works

1️⃣ Draw or remove bingo numbers using the controls below.
2️⃣ Set the current prize (title, description, image).
3️⃣ Click Export state.json and upload that file to the website.

The display will update automatically within a few seconds.

Draw Numbers

Click a number above to remove it.

Prize

The image will be embedded directly into the display (recommended).

Finish & Export

When you are done making changes, export the shared game state file.
